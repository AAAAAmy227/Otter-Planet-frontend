# Otter Planet Frontend

## 快速预览（本地 host）

这是一个纯静态页面工程，当前可直接预览「领水獭 Step 1」模块。

### 方式 1：使用脚本（推荐）

```bash
./scripts/preview.sh
```

默认端口是 `4173`，打开：

- http://localhost:4173

### 方式 2：指定端口

```bash
./scripts/preview.sh 8080
```

打开：

- http://localhost:8080

### 不使用脚本（等价命令）

```bash
python3 -m http.server 4173
```
